Configurar Google Drive
Conecte o portal à conta solarproportal@gmail.com
1. Google Drive API Key
A API Key permite que o portal leia os arquivos públicos do seu Drive sem login. Obtenha em console.cloud.google.com → APIs & Services → Credentials → Create Credentials → API Key. Depois restrinja à Google Drive API.
Esta chave ficará no arquivo js/drive.js do seu servidor
2. IDs das Pastas do Drive
No Google Drive (solarproportal@gmail.com), crie 3 pastas e compartilhe cada uma como "Qualquer pessoa com o link pode visualizar". Copie o ID da URL de cada pasta e cole abaixo.
Certificados
Inversores
Módulos
3. Nomenclatura dos arquivos
Para que o portal identifique a marca e o modelo automaticamente, use o padrão:

MARCA_MODELO_TIPO.pdf

Exemplos:
Growatt_MIN6000TL_Certificado.pdf
Deye_SUN6K_SG03LP1_Datasheet.pdf
Canadian_HiKu6_550W_Datasheet.pdf

Você também pode adicionar uma descrição ao arquivo no Drive (clique com botão direito → Detalhes → Descrição) para aparecer no card do portal.
Código gerado — cole em js/drive.js

        

      

    


    
    

      
Guia de uso

      
Como configurar e usar o SolarPro Portal


      

        
Passo a passo para colocar o site no ar

        

          

            
1

            

              
Criar conta no Google Cloud

              

                Acesse console.cloud.google.com com a conta solarproportal@gmail.com.
                Crie um novo projeto (ex: "SolarPro Portal").
                Ative a Google Drive API em APIs & Services → Library.
              

            

          

          

            
2

            

              
Gerar API Key

              

                Em APIs & Services → Credentials → Create Credentials → API Key.
                Após criar, clique em "Restrict Key" e selecione Google Drive API.
                Copie a chave gerada.
              

            

          

          

            
3

            

              
Criar pastas no Google Drive

              

                No Google Drive da conta solarproportal@gmail.com, crie 3 pastas:
                Certificados, Inversores e Módulos.
                Para cada pasta: botão direito → Compartilhar → "Qualquer pessoa com o link" → Visualizador.
                Copie o ID de cada pasta da URL.
              

            

          

          

            
4

            

              
Configurar o portal

              

                Acesse o painel Configurar Drive ao lado, insira a API Key e os IDs das pastas.
                Clique em "Salvar configuração" e copie o código gerado para o arquivo js/drive.js.
              

            

          

          

            
5

            

              
Hospedar o site

              

                Opção gratuita: GitHub Pages — crie um repositório público, suba todos os arquivos do portal e ative o GitHub Pages nas configurações.

                Ou use qualquer hospedagem de arquivos estáticos: Netlify, Vercel, Cloudflare Pages, ou hospedagem compartilhada convencional.
              

            

          

          

            
6

            

              
Adicionar documentos

              

                Para adicionar novos arquivos, basta fazer upload no Google Drive nas pastas correspondentes.
                O portal lê o Drive em tempo real — não é necessário alterar código.
                Nomeie os arquivos como MARCA_MODELO_TIPO.pdf para melhor organização.
